### 🌍 28/05/2026 — Sistema Multi-Idioma Global 🌐

<p>
  <img src="https://img.shields.io/badge/✅%20es.json-Español-10B981?style=flat-square&labelColor=065F46"/>
  <img src="https://img.shields.io/badge/✅%20en.json-English-3B82F6?style=flat-square&labelColor=1E3A8A"/>
  <img src="https://img.shields.io/badge/✅%20pt.json-Português-F59E0B?style=flat-square&labelColor=78350F"/>
</p>

El bot ahora habla en el idioma de cada usuario de forma automática. Sin configuración manual, sin mensajes hardcodeados — todo sale del JSON correcto según el idioma guardado en la base de datos.

**Lo que cambió:**
- ✅ Todos los plugins usan variables dinámicas en lugar de texto fijo en español
- ✅ Archivos `es.json`, `en.json` y `pt.json` unificados y completos
- ✅ Estructura centralizada: `plugins.nombre_del_plugin`
- ✅ Sistema escalable para agregar nuevos idiomas sin tocar el código
- ✅ Código más limpio, mantenible y consistente

**Cómo cambiar el idioma:**

```bash
.idioma es   # 🇦🇷 Español
.idioma en   # 🇺🇸 English
.idioma pt   # 🇧🇷 Português
```

El idioma se guarda automáticamente por usuario y todos los mensajes del bot se adaptan de inmediato.

### ⚡ ¿Cómo activarla?

Mencioná a Luna en el grupo con su `@tag` seguido de lo que necesitás. Sin prefijos como `.` o `/` — es una conversación real.

```
@Luna [lo que quieras]
```

### 🧠 Motor de IA Multi-Motor con Fallback Automático

Luna usa **múltiples motores de IA en cadena**. Si el primero falla o está ocupado, pasa automáticamente al siguiente sin que el usuario note nada. El orden se aleatoriza para distribuir la carga, y siempre hay un motor garantizado.

> Sin importar la hora ni la carga, Luna **siempre va a responder**.

### 🔍 Búsqueda Inteligente Multi-Fuente

Cuando le preguntás algo, Luna consulta **múltiples fuentes al mismo tiempo** y combina la mejor respuesta. Si es corta, la expande con IA antes de responder. Si es extensa y precisa, la manda directamente.

### 📋 Guía Paso a Paso — BoxMine

**Paso 1 — Crear cuenta**

Ingresá a [dash.boxmineworld.com/register](https://dash.boxmineworld.com/register), completá usuario, correo y contraseña. Confirmá tu correo.

---

**Paso 2 — Crear el servidor**

1. En el panel hacé clic en **Crear Servidor**
2. Poné un nombre *(ej: Botluna)*
3. En tipo seleccioná: **Software → Juegos → WhatsApp-bot**
4. En especificaciones elegí: **WhatsApp bot (Luna-Botv6)**
5. En ubicaciones elegí **Free-N6** *(nodo free recomendado)*
6. Deslizá hacia abajo y seleccioná **Crear Servidor**

> [!TIP]
> Si Free-N6 no tiene lugar disponible, probá con **Free-N4** o **Free**. El nodo N6 suele ser el más estable.

---

**Paso 3 — Entrar al servidor y vincular WhatsApp**

1. En tu lista de servidores hacé clic en **Gestionar**
2. Dentro del panel tocá las **tres líneas** (☰) arriba a la derecha
3. Seleccioná **Consola**
4. Elegí `start` y esperá que se descarguen todas las dependencias

Una vez que termina la instalación verás:

```
Iniciando Luna-Botv6...
Elegí un método para vincular:
1. Código QR
2. Código de 8 dígitos
```

5. Escribí `1` para QR o `2` para código de 8 dígitos
   - Si elegís **2**, ingresá tu número sin espacios *(ej: `+5493483511079`)*
   - El bot te enviará un código de 8 dígitos a tu WhatsApp
   - Ingresalo en WhatsApp: **Configuración → Dispositivos vinculados → Vincular con número**

> [!IMPORTANT]
> En la consola de BoxMine **solo escribís números** para elegir opciones al vincular. No uses `npm start` — el panel lo gestiona todo automáticamente.

### 🟢 Requisitos

| Requisito | Detalle |
|:----------|:--------|
| 💻 Sistema | Windows 10 / Windows 11 |
| 🌐 Conexión | Internet activa |
| 🔑 Permisos | Administrador en el equipo |

---

### 🟡 Paso 1 — Instalar Git

Descargar Git desde el sitio oficial: **https://git-scm.com/download/win**

Instalá con todas las opciones por defecto. Asegurate de marcar **"Add Git to PATH"**.

Verificar instalación:
```bash
git --version
```
✅ Si muestra una versión, está correcto.

---

### 🔵 Paso 2 — Instalar Node.js

Descargar Node.js LTS desde: **https://nodejs.org/**

- Abrí el `.msi` descargado → Siguiente → Siguiente → Aceptar
- ⚠️ Marcá: **"Add to PATH"** (IMPORTANTE)
- Finalizá la instalación

Verificar:
```bash
node -v
npm -v
```
✅ Si muestra versiones, está correcto.

---

### 🟠 Paso 3 — Arreglar PowerShell

Abrí **PowerShell como Administrador** y ejecutá:

```powershell
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
```

Cuando pregunte, escribí `Y` y presioná Enter.

---

### 🟣 Paso 4 — Instalar Chocolatey

En **PowerShell como Administrador** ejecutá:

```powershell
Set-ExecutionPolicy Bypass -Scope Process -Force; `
[System.Net.ServicePointManager]::SecurityProtocol = `
[System.Net.ServicePointManager]::SecurityProtocol -bor 3072; `
iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
```

> Reiniciá PowerShell después de instalar.

Verificar:
```bash
choco -v
```
✅ Si muestra la versión, está correcto.

---

### 🔴 Paso 5 — Instalar Dependencias

```bash
choco install python3 ffmpeg imagemagick -y
```

---

### 📁 Paso 6 — Clonar el Proyecto

```bash
git clone https://github.com/Luna-botv6/Luna-Botv6-Project.git
cd Luna-Botv6-Project
```

---

### ⚙️ Paso 7 — Instalar el Bot

```bash
npm install
```

---

### 🛠️ Errores Comunes en Windows

```bash
npm uninstall sharp
npm install sharp --os=win32 --cpu=x64
npm rebuild sharp
```

</details>

<details>
<summary>❌ Error con <code>tmp</code></summary>

```bash
mkdir C:\tmp
```

</details>

<details>
<summary>❌ Error de permisos en npm</summary>

```powershell
Set-ExecutionPolicy -Scope Process Bypass
```

</details>

```bash
npm start
```

> Los `npm warnings` y `vulnerabilities` son normales y no bloquean el bot.
